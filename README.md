# 🛡️ SOC Home Lab

A hands-on cybersecurity home lab built to develop practical blue team and SOC analyst skills.

## 🔧 Lab Setup
- **Virtualization:** Oracle VirtualBox
- **Attacker/Analyst Machine:** Kali Linux 2026.1
- **SIEM:** Splunk Enterprise 10.2.1

## 🎯 Objectives
- Collect and analyze logs using Splunk
- Detect suspicious activity and investigate alerts
- Build practical skills for a Junior SOC Analyst role

## 📁 Structure
- `/setup` - Lab setup documentation with screenshots
- `/investigations` - Detection and investigation write-ups
- `/screenshots` - Evidence and dashboard screenshots
